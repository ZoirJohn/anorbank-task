# Приложение для сравнения продуктов

Небольшое React-приложение, позволяющее выбирать товары и сравнивать их между собой.

## Обзор

Приложение отображает список товаров, заданных вручную (hardcoded). Пользователь может выбрать несколько товаров для сравнения. Выбранные товары сохраняются и остаются доступными даже после обновления страницы благодаря `localStorage`.

## Возможности

- Список товаров, заданных вручную
- Выбор товаров для сравнения
- Сохранение выбранных товаров в `localStorage`
- Восстановление состояния после перезагрузки страницы
- Сервис, имитирующий работу store (логика хранения данных)
- Использование `Outlet` для вложенных маршрутов в React Router

## Архитектура

- **Service layer** — имитирует backend/store и управляет выбранными товарами  
- **localStorage** — сохраняет состояние выбора между сессиями  
- **Routing** — используется React Router с `Outlet` для layout-структуры  
- **State management** — состояние синхронизируется между сервисом и UI  

## Как это работает

1. Товары берутся из статического массива  
2. Пользователь выбирает товары для сравнения  
3. Выбранные товары сохраняются:
   - в сервисе (in-memory store)
   - в `localStorage` (для сохранения после обновления страницы)
4. После перезагрузки страницы состояние восстанавливается  
5. Страница сравнения обновляется автоматически при изменении выбора
